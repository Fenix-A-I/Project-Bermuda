To install Caddy:

1. /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. brew install caddy

3. check version (optional) - caddy version

4. sudo caddy start --config /path/to/your/Caddyfile
    for us it's caddy/Caddyfile
